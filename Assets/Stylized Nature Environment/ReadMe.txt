1.Go to asset store, download "Post Processing Stack", then import into your project.

2.Add the "post-processing behaviour" component in the Main Camera.

3.Then, drag a sample Post-Processing Profile in the box.
(Under the main folder, you can find 2 preset in a folder named "PostProcessing Sample", Sping and Autumn presets.........)

4.Don't forget go to"Edit-Project Settings-Player",change the "Color Space" from Gama to Linear.
